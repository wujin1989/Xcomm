# 概述
Xcomm 是一个集成了多种通信协议的客户端工具包，将不同协议的功能封装到一个统一且精简的接口中供开发者使用。无论是工业协议还是其他通用通信协议，Xlink 都提供了无缝的支持，简化了复杂环境中的数据交互和系统集成。通过 Xlink，用户可以高效地实现跨平台和跨协议通信，显著提高开发效率和系统兼容性。

# 特性
- 跨平台兼容性

# 要求
编译器需要支持 C11 标准。


# 编译
    cmake -B out
    cmake --build out --config Release -j 8
    cmake --install out
